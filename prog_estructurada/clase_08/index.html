Clase del día Jueves 19 y 26 de Mayo y 2 de Junio

Programas hechos:

  1. Realizar un programa que ayude a la gestion de un supermercado. (Utilizando funciones y procedimientos)
  2. Realizar un programa que permita realizar las operacioes básicas de matemática. (Utilizando funciones sin parametros ni retornos)
  3. Realizar un programa que permita realizar las operaciones básicas de matemática. (Utilizando funciones con parámetros y retornos)
  4. Del programa anterior, agregarle funciones para determinar si el resultado es par o impar y determinar cuál de los numeros es mayor o menor.
